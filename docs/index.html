Simple Grid

Simple Grid is a simple and responsive grid system, initially containing a system of 8 columns, and can be changed through the use of sass.

  • Getting Started
  • Grid
  • Responsive classes
  • SASS

Getting Started

Download the file simple-grid.css or simple-grid.min.css, copy to your project and linking in header of the page:

<link rel="stylesheet" href="path/simple-grid.css" />

Display Classes

Display classes let you control the display of columns on certain screen sizes

ClassDescription
hide-on-lgHide the collumn in large sizes
hide-on-mdHide the collumn in medium sizes
hide-on-smHide the collumn in small sizes
hide-on-exHide the collumn in extra small sizes
hide-on-mnHide the collumn in minimun sizes
show-on-lgShow the collumn only in large sizes
show-on-mdShow the collumn only in medium sizes
show-on-smShow the collumn only in small sizes
show-on-exShow the collumn only in extra small sizes
show-on-mnShow the collumn only in minimun sizes
